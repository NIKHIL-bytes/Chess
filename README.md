# ♟️ Chess

A fun little chess game I built using **HTML, CSS, and JavaScript** while hanging out with a friend and trying to kill some boredom.

What started as a random idea quickly turned into a **beautiful, vibrant neon-themed chess board** made just for the two of us to play and enjoy.

The project focuses on:
- ✨ A glowing neon UI
- 🎨 Vibrant and colorful design
- ♟️ Classic chess gameplay
- 💻 Built purely with HTML, CSS, and JavaScript
- 🎯 Made for fun, creativity, and practice

Sometimes the best projects are the unplanned ones.

This one was simply created for fun, but it turned out to be something I genuinely loved building.
